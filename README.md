# Her Birthday Universe

A cinematic birthday surprise website built with React, Vite, Tailwind CSS, Framer Motion, GSAP, looping background music, interactive memories, hidden unlocks, a mini game, and a final birthday reveal.

## Edit The Surprise

Most personalized content lives in:

`src/data/siteConfig.js`

Update her name, birthday date, messages, timeline memories, reasons, gallery images, unlock answers, final reveal text, and background song there.

## Run Locally

```bash
npm install
npm run dev
```

## Background Music

The background song lives in:

`public/audio/happy-birthday.mp3`

To use another song, replace that file or update `localFallbackTrack.src` in `src/data/siteConfig.js`.
